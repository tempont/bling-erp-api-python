# Referência da API

Esta seção documenta automaticamente todos os recursos e métodos do SDK.

As páginas são geradas a partir dos docstrings Google-style do código-fonte
pelo `mkdocstrings`. Cada página lista a assinatura dos métodos, parâmetros,
tipos e valores de retorno com formatação colorida.
